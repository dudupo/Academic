\begin{abstract}
  We suggest a fault tolerance scheme in the circuit model that both uses the 4D Toric code and has a constant-depth correction cycle. Until now, in the circuit model, the use of a topological code as the primary encoding and a constant-depth correction cycle were achieved separately; this is the first work to achieve them simultaneously. In addition, to date, the 4D-Toric code has been proposed as a memory element in the thermal noise model. Our work shows that it is also suitable as a memory in the circuit model. To show that for the correction cycle we adopt the sweep rule defined in \cite{Kubica_2019}, and hence bypass the need for the hierarchical correction cycle, the proof is based on alternative proofs for Gacs theorems \cite{Berman}, \cite{gacs2021newversiontoomsproof}, which are distinguished from the standard approaches as they trace the past that lead to error clusters.
\end{abstract}
